Instagram Reach Analysis

Project Overview
Instagram is one of the most widely used social media platforms for content creators, businesses, bloggers, and influencers. However, Instagram frequently updates its algorithm, which directly affects the reach and engagement of posts. This project uses Data Science and Machine Learning techniques to analyze Instagram post performance and identify factors that influence post reach. By analyzing metrics such as likes, shares, comments, saves, profile visits, and hashtag performance, this project helps understand what drives higher impressions and engagement.

Objectives
Analyze Instagram post reach using data collected from posts.
Understand how different engagement metrics affect impressions.
Visualize the distribution of impressions from different sources.
Extract insights from captions and hashtags.
Build a Machine Learning model to predict Instagram impressions.

Dataset
The dataset contains 119 Instagram posts with the following features:

Feature	Description
Impressions	Total number of times the post was viewed
From Home	Impressions from the home feed
From Hashtags	Impressions from hashtags
From Explore	Impressions from explore page
From Other	Impressions from other sources
Saves	Number of saves
Comments	Number of comments
Shares	Number of shares
Likes	Number of likes
Profile Visits	Profile visits generated by the post
Follows	Followers gained from the post
Caption	Caption text of the post
Hashtags	Hashtags used in the post
